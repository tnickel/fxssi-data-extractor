# API-Quelle: https://production.dataviz.cnn.io/index/fearandgreed/graphdata
# Symbol: BTC/USD
# 
# Signal-Logik (Contrarian-Ansatz):
# - Index 0-44 (Fear/Extreme Fear)    → BUY Signal (Long wenn andere Angst haben)
# - Index 45-55 (Neutral)             → NEUTRAL Signal (Seitwärts)
# - Index 56-100 (Greed/Extreme Greed)→ SELL Signal (Short wenn andere gierig sind)
#
# Mapping auf CurrencyPairData:
# - buyPercentage = 100 - fearGreedIndex (bei Fear ist Buy hoch)
# - sellPercentage = fearGreedIndex (bei Greed ist Sell hoch)
#
# Beispiele:
# - Fear Index = 20:  Buy=80%, Sell=20%, Signal=BUY
# - Neutral Index = 50: Buy=50%, Sell=50%, Signal=NEUTRAL
# - Greed Index = 80: Buy=20%, Sell=80%, Signal=SELL
